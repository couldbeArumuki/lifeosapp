# LifeOS - Personal Activity & Progress Tracker

A beautiful personal dashboard for tracking your life's progress. No authentication required — built for personal use with a public read-only view.

## 🎨 Features

- 📊 **Dashboard** — Daily stats, charts, quick actions
- ✅ **Tasks** — To-do list with priorities and status
- 🎯 **Habits** — Daily habit tracker with streaks
- 🇯🇵 **Japanese Learning** — JLPT vocabulary & kanji tracker
- 📝 **Study Log** — Log study sessions by subject
- 📊 **Trackers** — Mood, sleep, health and finance tracking
- 🏆 **Goals** — Goal setting with progress bars
- 📈 **Analytics** — Visual insights and trends
- 🌐 **Public Profile** — Read-only shareable view
- 🌙 **Dark Mode** — Full dark mode support
- 📱 **Responsive** — Mobile-first design

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## 🛠️ Tech Stack

- **React 18** + Vite
- **Tailwind CSS** with custom design system
- **React Router v6** for navigation
- **Recharts** for data visualization
- **Lucide React** for icons
- **localStorage** for data persistence
